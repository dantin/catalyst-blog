### 快捷键

```console
Ctrl + J，显示JavaDoc
Ctrl + H，类的层次结构
CMD + F，查找
CMD + F12，查看类中全部方法和属性
Ctrl + Alt + H，查看方法调用链
CMD + Alt + U，查看类图
Ctrl + H，类的继承链

CMD + [+/-]，展开/收起方法
CMD + Shift + [+/-]，全部展开/收起

查看类图，右键 --> Diagrams --> Show Diagram...
```
